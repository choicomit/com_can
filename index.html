AI 기반 국가기술자격 기출문제 학습 시스템

문제 풀이 → 자동 채점 → 오답 복습 → 유사 문제 반복 학습 흐름으로, 시험 직전까지 필요한 문항을 빠르게 마스터하세요.

오답 중심 복습

틀린 문항을 자동으로 누적하여 재학습 우선순위를 정리합니다.

유사 문제 반복

핵심 키워드를 기반으로 유사 문항을 제공하여 약점을 보완합니다.

빠른 이동

문항 번호 클릭/입력으로 원하는 문제로 즉시 이동할 수 있습니다.

학습 효율

불필요한 반복을 줄이고 필요한 문항에 집중하도록 구성했습니다.

자격증 선택
현재는 컴퓨터활용능력 2급만 이용할 수 있습니다.
관리자용
팁: ESC로 도움말 닫기

오늘의 전략

1) 전체 문제를 빠르게 훑어보며 출제 유형 파악

2) 오답 문제만 2~3회 반복 학습

3) 유사 문제로 약점 유형 보완

합격을 기원합니다 ✨

사용 방법

  • 자격증 선택에서 과목을 선택하면 문제 화면으로 이동합니다.
  • 문제 화면에서는 자동 채점과 오답 추적, 유사 문제 기능을 이용할 수 있습니다.
  • 문항 번호 클릭/입력으로 빠른 이동이 가능합니다.
